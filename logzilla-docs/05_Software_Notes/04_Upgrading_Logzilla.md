# Upgrading LogZilla

To upgrade your version to the last, run the following command:

```
sudo logzilla upgrade
```

Note that you can upgrade multiple versions at once, without having to do
the intermediate versions. For example, if you are on 6.30.0 and the current
master is 6.33, `logzilla upgrade` will upgrade you to 6.33.


## End of Life / End of Support Versions

LogZilla v6.26.0 and above are currently supported versions. Earlier versions are
end of life and need to be upgraded.
